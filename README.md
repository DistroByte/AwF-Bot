# AwF-Bot
A bot that helps communicate between Factorio clients and a Discord server

## Installation
For installation, see [INSTALLATION.md](INSTALLATION.md). The bot can be self-hosted.

The bot uses [bisa/factorio-init](https://github.com/bisa/factorio-init) to start, stop etc. the servers. It uses discord.js v12 to communicate. To see the bot in action, please visit the [discord](http://awf.yt)
